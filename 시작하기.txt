✨ 자동으로 다음이 실행됩니다:

  3. http://127.0.0.1:7860 에서 사용 가능

🎯 기본 사용 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  마네킹 이미지 업로드 (필수)
   - 파일을 드래그&드롭 하거나 클릭하여 선택

2️⃣  카테고리 선택
   - 일반의류: 상의 + 하의 따로 업로드
   - 원피스: 한 벌로 업로드

3️⃣  의류 이미지 업로드
   - 배경이 자동으로 제거됩니다!
   - 배경이 없는 이미지도 OK

4️⃣  필요하면 미세 조정
   - 슬라이더 또는 숫자 입력으로 조정
   - 자동 맞춤 버튼으로 한 번에!

5️⃣  결과 다운로드
   - "결과 이미지 다운로드" 클릭
   - 고해상도 PNG로 저장됨

💡 팁
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 배경 제거가 정확하지 않으면?
  → 배경을 더 단색으로 만들기 (흰색/회색/검은색이 최고)
  → 조명 개선 (의류와 배경의 색상 대비 강화)

✓ 위치/크기 조정이 어렵다면?
  → 슬라이더 옆의 숫자 입력칸에 직접 값 입력
  → "자동 맞춤" 버튼으로 최적값 자동 설정

✓ 처음부터 다시 시작하려면?
  → "초기화" 버튼 클릭
  → 모든 조정이 초기화됨

   → 배경이 가능한 단색인 이미지 사용
   → 매우 복잡한 패턴의 배경은 정확도 저하
