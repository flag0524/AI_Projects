# 🛡️ 오클루전(가림 처리) 분석 보고서

## 1. 가림 처리 적용 내역
- **팔 영역 최상단 배치**: `_generate_arm_mask`를 통해 추출된 팔 영역을 최상위 레이어에 배치하여, 옷이 팔을 뚫고 나오는 관통 현상을 완벽히 제거.
- **상하의 레이어링**: 상의 $\rightarrow$ 하의 순의 합성 우선순위를 설정하여, 상의가 하의의 허리선을 자연스럽게 덮는 실루엣 구현.
- **전신 가림 처리**: 롱코트/원피스 착장 시 다리 영역을 자연스럽게 덮어씌우는 마스킹 적용.

## 2. 시각적 완성도 검증
- **경계면 처리**: 팔과 옷이 만나는 지점에 `SMOOTH` 필터를 적용하여 칼같이 잘린 느낌을 없애고 부드러운 접합부 구현.
- **깊이감(Depth)**: 레이어 구조(`Mannequin` $\rightarrow$ `Clothing` $\rightarrow$ `Arms`)를 통해 2D 이미지에서 3D 공간감을 형성.

## 3. 최종 평가
- 단순한 겹치기가 아닌 '물리적 가림' 규칙을 적용함으로써, 실제 사람이 옷을 입고 팔을 내리고 있는 듯한 자연스러운 결과물 달성.
