# 🌓 뎁스 및 섀도우 렌더링 분석 보고서

## 1. 조명 일치성 검증 (Lighting Consistency)
- **광원 분석**: 마네킹의 원본 명암 맵(Luminance Map)을 추출하여 의류 레이어에 1:1 매핑.
- **결과**: 의류의 하이라이트와 섀도우 방향이 마네킹의 조명 방향과 일치하여, 합성 티가 나지 않는 자연스러운 렌더링 달성.

## 2. 음영 처리 상세 (Shadowing)
- **접촉면 음영**: 의류와 신체가 만나는 경계 영역에 미세한 앰비언트 오클루전(AO) 효과를 부여하여 '떠 있는 느낌'을 완전히 제거.
- **무결성 유지**: AI에 의한 임의의 그림자 생성을 배제하고, 오직 마네킹 원본 조명 데이터를 기반으로 한 수학적 연산으로만 처리.

## 3. 최종 퀄리티 평가
- **입체감**: 단순한 2D 합성이 아닌, 조명과 뎁스가 반영된 2.5D 렌더링 결과물 생성.
- **자연스러움**: 배경 그림자와 의류 내부 음영이 조화를 이루어 실제 스튜디오 촬영 컷과 유사한 퀄리티 확보.
