# 👕 상의 피팅 정밀 분석 보고서

## 1. 적용된 피팅 규칙 검증
- **어깨 라인 정렬**: 마네킹 분석 데이터(`shoulder_w`)를 기반으로 상의 너비를 매핑하여 어깨선 일치 확인.
- **넥라인 연결**: `neck_y` 좌표를 기준으로 상단 정렬하여 자연스러운 연결 구현.
- **왜곡 방지**: `Aspect-Ratio` 보존 로직을 통해 로고, 패턴, 텍스트의 찌그러짐(Stretch) 현상 제거.
- **질감 유지**: `LANCZOS` 보간법을 사용하여 원단 질감 및 고해상도 엣지 유지.

## 2. 기술적 구현 사항
- **Adaptive Scaling**: $\text{Target Width} \rightarrow \text{Original Ratio} \rightarrow \text{Target Height}$ 순의 계산 체계를 통해 원본 디자인 훼손을 원천 차단.
- **Sizing Constraint**: 상체 최대 높이 제한(`max_h`)을 설정하여 의류가 하의 영역을 과하게 침범하는 현상 방지.

## 3. 최종 평가
- 상의의 원본 실루엣이 유지되면서 마네킹 체형에 자연스럽게 안착됨.
- 특히 로고와 프린트 부분이 왜곡 없이 선명하게 표현됨.
