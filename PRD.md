# LaonGEN — 제품 요구사항 문서 (PRD)

**제품명:** LaonGEN (라온젠) — 마네킹/제품 사진 기반 모델 착용 컷 생성
**버전:** 2.1.0
**최종 수정:** 2026-06-05
**저장소:** https://github.com/flag0524/Fashiion_TryOn

## 1. 제품 개요

마네킹·옷걸이에 걸린 **제품 사진만으로** 실제 모델이 착용한 **자연스러운 패션 컷**을 생성하는 웹 서비스.

쇼핑몰·브랜드가 모델 촬영 없이 의류 제품 사진(상의/하의/원피스)을 업로드하면, AI가 실제 모델이 착용한 이미지를 생성한다.

### 1.1 배경 및 문제 정의
- 의류 모델 촬영은 비용·시간이 크다 (스튜디오, 모델, 사진작가).
- 마네킹에 옷을 2D로 합성하는 절차적 방식은 "유령처럼 겹쳐 보이는" 부자연스러운 결과를 낸다 (프로젝트 초기 검증에서 확인).
- 자연스러운 착용 모습은 **생성형 AI(diffusion 기반 try-on)**로만 가능하다.

### 1.2 핵심 가치
- **무료 운영 가능**: GPU 없이, 무료 HuggingFace Spaces로 상업 품질 생성.
- **간편함**: 제품 사진만 업로드 → 자연스러운 모델 컷.
- **풀코디**: 상의 + 하의 동시 착용 지원.

## 2. 사용자 및 사용 시나리오

| 사용자 | 시나리오 |
|--------|----------|
| 온라인 쇼핑몰 운영자 | 마네킹 제품컷을 모델 착용 컷으로 변환해 상세페이지에 사용 |
| 패션 브랜드 MD | 신상품 룩북 시안을 빠르게 생성 |
| 1인 셀러 | 모델 섭외 없이 SNS 홍보용 착용 이미지 확보 |

### 2.1 사용자 플로우
1. 웹 접속 → **"모델 컷 생성"** 모드 선택 (기본값)
2. (선택) 원하는 모델 템플릿 업로드 — 미선택 시 기본 중립 모델 사용
3. 의류 추가: 상의/하의/원피스/액세서리 (복수 가능)
4. **"모델 컷 생성"** 클릭
5. 약 1분 후 자연스러운 모델 착용 컷 표시 → 다운로드

## 3. 기능 요구사항

### 3.1 핵심 기능 (구현 완료)
- [x] 제품 사진 업로드 (상의/하의/원피스/액세서리, 복수 동시)
- [x] **모델 컷 생성** — 실제 모델 착용 컷 (Leffa try-on diffusion)
- [x] **상의 + 하의 풀코디** 순차 착용
- [x] 모델 템플릿 선택 (사용자 업로드 또는 기본 중립 모델)
- [x] 결과 이미지 미리보기 + PNG 다운로드
- [x] **마네킹 피팅** 보조 모드 (절차적 2D 합성, 폴백용)
- [x] 생성 백엔드 상태 표시 (AI 연결 여부)

### 3.2 두 가지 운영 모드
| 모드 | 방식 | 품질 | 용도 |
|------|------|------|------|
| **모델 컷 생성** (기본) | 생성형 AI (Leffa) | 자연스러움 ★★★ | 실제 서비스 결과물 |
| 마네킹 피팅 | 절차적 2D 합성 | 제한적 ★ | AI 불가 시 폴백 |

### 3.3 입력/출력
- **입력:** JPEG/PNG 의류 이미지 (흰 배경 권장), 최대 10MB
- **의류 타입:** `top` / `bottom` / `dress` / `accessory`
- **출력:** PNG 모델 착용 컷 (base64), 처리 시간·생성 방식(method) 포함

## 4. 비기능 요구사항

| 항목 | 목표 | 현황 |
|------|------|------|
| 생성 시간 (상의+하의) | 1~2분 | ~60–70초 (검증) |
| 운영 비용 | 무료 | 무료 HF 토큰 (ZeroGPU 일일 할당량) |
| GPU 의존성 | 불필요 | 클라우드 Space에 위임 |
| 결과 해상도 | 768×1024 | Leffa 기본 출력 |

### 4.1 제약사항
- **마네킹에 직접 착용 불가:** try-on AI는 사람 포즈·체형(DensePose) 감지가 필요해, 마네킹(머리·다리 없음)에는 적용 불가. 따라서 의류를 **실제 모델 템플릿**에 입히는 방식이 유일한 자연 생성 경로.
- **무료 할당량:** HF ZeroGPU 무료 일일 한도 존재 (소진 시 ~24시간 후 리셋, 또는 HF PRO 구독).
